Общая информация

Требует установленного приложения РМК версии 1.199.16 или выше. Версии РМК 1.202.0-1.204.4 имеют проблемы совместимости с SDK.

API представляет собой отправку Intent сообщений, на которые подписано приложение ru.aqsi.cashierworkplace

В ответ, приложение ru.aqsi.cashierworkplace отправляет информацию о результате операции приложению, использующему библиотеку commons-x.x.x.aar

Подключение библиотеки aQsi commons Инструкция по подлкючению библиотек AAR в Android Studio

Процесс пробития чека

В среде aQsi чек представляет собой набор номенклатуры и дополнительных параметров для формирования фискального документа.

Перед выполнением фискальных операций нужно убедиться, что в кассу aQsi произведён вход под кассиром.

Проверить активного кассира можно с помощью метода getActiveCashierUuid. Если кассира нет, то его надо выставить из базы кассиров - CashierDb. Если необходимого кассира нет в базе кассиров, необходимо добавить его в базу методом createCashier.
