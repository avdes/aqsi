fiscalizeCheque

fun fiscalizeCheque(context: Context, type: ChequeType, positions: ArrayList<ChequePosition>, chequeClose: ChequeClose, clientContact: String? = null, clientName: String? = null, clientInn: String? = null, settlementAddress: String? = null, settlementPlace: String? = null, additionalAttribute: String? = null, additionalUserAttribute: AdditionalUserAttribute? = null, clientBirthDate: String? = null, clientNationality: String? = null, clientDocType: String? = null, clientDocNumber: String? = null, clientAddress: String? = null, industryAttribute: IndustryAttribute? = null, operationalAttribute: OperationalAttribute? = null, callback: AqsiDataResultReceiver.AqsiCallback<Cheque>)

Фискализирует чек. Чек должен содержать минимальное число полей, необходымых для фискализации, такие как позиции, система налогообложения список оплат по чеку. Подробнее об этих полях можно посмореть в документации

Parameters

type

Тип чека

positions

Позиции чека

chequeClose

Информация о закрытии чека

callback

Обработчик результата фискализации чека. В качестве ответа будет возвращен чек с фискальными параметрами.

clientContact

Контакт клиента. Email или телефон клиента, используется для электронного чека.

Должен соответствовать формату указанному в ФФД (+{Ц} или {С}@{С}), тэг 1008
