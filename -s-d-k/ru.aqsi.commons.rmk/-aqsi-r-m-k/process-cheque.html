processCheque

@JvmStatic
fun processCheque(context: Context, chequePositions: ArrayList<ChequePosition>, callback: AqsiDataResultReceiver.AqsiCallback<Cheque>, params: ChequeProcessingParams? = null)

Пробить чек с помощью кассы Акси. Товары передаются в виде массива моделей товара. Для пробития чеков рекоментуется использовать этот метод. Метод может работать некорректно при использовании Proguard.
