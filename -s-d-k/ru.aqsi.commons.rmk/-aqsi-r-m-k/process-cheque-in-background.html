processChequeInBackground

@JvmOverloads
@JvmStatic
fun processChequeInBackground(context: Context, positions: ArrayList<ChequePosition>, callback: AqsiDataResultReceiver.AqsiCallback<Cheque>, params: ChequeProcessingParams? = null)

Пробить чек с помощью кассы Акси. Товары передаются в виде массива моделей товара. Это более простой метод, который просто печатает чек, без перекидывания в графический интерфейс aQsi РМК. В этом случае всю логику проведения операции оформления чека следует реализовать на стороне вашего приложения. Обратите внимание на более простой метод, где логика проведения операции оформления чека реализована на стороне приложения РМК: processCheque Метод может работать некорректно при использовании Proguard.

Parameters

context

контекст вызова метода

positions

массив товаров, на которые необходимо пробить чек возврата

callback

callback, в случае успеха возвращает параметры фискального чека Cheque
