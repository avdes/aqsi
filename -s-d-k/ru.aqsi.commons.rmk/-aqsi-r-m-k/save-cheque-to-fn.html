aqsiDataPrintCallBack

В случае оплаты по карте возвращает ответ эквайрингового ядра в XML. Описание параметров XML необходимо запросить дополнительно либо, по возможности, понять на основе названий тэгов. Также метод возвращает данные о последнем ФД в зависимости от значения флага fastModeWithoutDataInCallback, если fastModeWithoutDataInCallback = true, то метод вернет в каллбек aqsiDataPrintCallBack модель ParamsChequePrint у которой будут заполнены только поля ParamsChequePrint.fd и ParamsChequePrint.fp, остальные поля будут либо пустые либо null. Если не устанавливать значение fastModeWithoutDataInCallback или установить его значение false, то метод вернет необходимую информацию о ФД.
