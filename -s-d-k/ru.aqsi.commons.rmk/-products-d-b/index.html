object ProductsDB

Класс предоставляет доступ к базе товаров, поддерживает следующие операции:

  • Найти конкретный товар по id getProduct

  • Создать новый товар createProduct

  • Обновить существующий товар updateProduct

  • Удалить существующий товар deleteProduct
