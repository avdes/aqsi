startAcquiringOperation

@JvmOverloads
@JvmStatic
fun startAcquiringOperation(context: Context, price: Long? = null, transactionType: String = Transaction.PURCHASE.value, rrn: Long? = null, number: Int? = null, transactionID: String? = null, resultReceiverCallback: AqsiDataResultReceiver.AqsiCallback<AcquiringData>)

Совершить эквайринговую операцию. Универсальный метод, подходит для любого типа эквайринга на кассе (встроенный эквайринг, Сбербанк upos, пинпад 2can). Для работы с Эквайринговым ядром Сбербанка (upos) рекомендуется использовать методы для прямой интеграции. Вопросы прямой интеграции с ПО Сбербанка нужно обсуждать напрямую с представителями Сбербанка.

Примечание: Сбербанк не поддерживает операцию Transaction.VOID
