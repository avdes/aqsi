ActiveCashierNotFoundException
Link copied to clipboard
class ActiveCashierNotFoundException : AqsiException

Исключение возникает, когда на кассе отсутствует активный кассир, но выполнение метода невозможно без наличия кассира. Для избежания ошибки необходимо убедится, что выставлен активный кассир, проверить это можно с помощью метода ru.aqsi.commons.rmk.ActiveCashierDB.getActiveCashierUuid в случае, если кассира нет, метод вернет null. Установить кассира можно с помощью метода ru.aqsi.commons.rmk.ActiveCashierDB.setActiveCashier
