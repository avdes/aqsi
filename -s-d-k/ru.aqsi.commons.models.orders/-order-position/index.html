quantityPerUnit
Link copied to clipboard
var quantityPerUnit: Double? = null

Количество предмета расчета, принимаемое за единицу (максимум 3 знака после запятой). Общее количество предмета расчета в позиции = quantity / quantityPerUnit. Значение по-умолчанию 1.0 будет приниматься в случае отсутствия значения или <= 0. Для случая дробного количества маркированного товара домножается на 10^3 и передаётся в виде целого числа в тэг 1294 (знаменатель).
